	Installation

1. Put "framework" directory at your domain or keep it somewhere
2. And in your Command Line or Consol type something like: php path\to\yii\framework\yiic.php webapp path\to\www\ApplicationName
 
Note: you can keep "framework" directory and "\ApplicationName" at the same place/domain (ex.: php c:\xampp\htdocs\domain\framework\yiic.php webapp c:\xampp\htdocs\domain)
